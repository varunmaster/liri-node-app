# liri-node-app

liri-node-app is a command line application that takes in parameters and returns data back from API response.

## Functions Available:

- concert-this
- spotify-this-song
- movie-this
- do-what-it-says

## Technologies Used

- Node.JS
- Axios
- Spotify API
- OMDb API
- Bands in Town API
